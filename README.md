# Little-Calendar

A simple **desktop calendar application** built with **HTML, CSS, JavaScript, and ElectronJS**.  
This project is part of my first journey into building an **Electron app**.

## Features

- Display current day, date, and month.
- Responsive and minimal design.
- Uses the **Pixeboy** custom font for a retro pixel-art style.
- Fully desktop-based using **ElectronJS**.

## Technologies Used

- **HTML5 & CSS3** – Markup and styling.
- **JavaScript (ES6+)** – Application logic.
- **ElectronJS** – Desktop app wrapper.
- **Node.js & npm** – Dependency management.
